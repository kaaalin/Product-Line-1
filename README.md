# Kendo Agentic UI Composer Prototype

Clickable Vite + React frontend prototype using the warm slide palette.

## Run locally

```bash
npm install
npm run dev
```

## Test

```bash
npm test
```

## Build

```bash
npm run build
```

## Publish on Vercel

1. Upload/push this whole folder to a GitHub repository.
2. In Vercel, choose **Add New Project**.
3. Import the GitHub repository.
4. Keep the detected framework as **Vite**.
5. Build command: `npm run build`.
6. Output directory: `dist`.
7. Deploy.
